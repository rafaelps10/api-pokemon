# Pokémon API 🐱‍👤

Uma API RESTful desenvolvida em **JavaScript (Node.js)** para consulta e gerenciamento de informações sobre Pokémon.  
Este projeto foi criado com foco em **Back-end**, aplicando boas práticas de desenvolvimento e arquitetura de sistemas.

---

## 🚀 Funcionalidades
- Listar todos os Pokémon disponíveis.
- Buscar Pokémon por **ID** ou **nome**.
- Retornar detalhes como **tipo**, **habilidades** e **estatísticas**.
- Estrutura modular e escalável, pronta para integração com aplicações front-end.

---

## 🛠️ Tecnologias Utilizadas
- **Node.js**  
- **Express.js** (para criação de rotas HTTP)  
- **JavaScript ES6+**  
- **JSON** para troca de dados
